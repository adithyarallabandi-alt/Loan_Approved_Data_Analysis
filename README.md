# Loan_Approved_Data_Analysis
Data analytics project showcasing Loan Approved Data Analysis using Python ,SQL ,Power BI.
Bank Loan Approval Data Analysis
Overview

This project demonstrates an end-to-end data analytics workflow focused on analyzing bank loan approval data. The goal is to uncover patterns and key factors that influence loan approval decisions and present insights through dashboards, reports, and presentations.

The project highlights practical experience with Python, SQL, Power BI, and business reporting, following real-world data analyst workflows.

Dataset

The dataset contains structured data related to bank loan approvals.

It includes information such as:

Applicant demographics

Income and employment details

Loan amount and loan term

Credit history

Loan approval status

The data is used for exploratory analysis, cleaning, SQL querying, and visualization.

Tools & Technologies

Python – Data loading, EDA, and data cleaning

Pandas, NumPy, Matplotlib, Seaborn

SQL – Data querying and analysis

PostgreSQL / MySQL / SQL Server

Power BI – Interactive dashboard creation

Gamma – Presentation (PPT) creation

Git & GitHub – Version control and project documentation

Project Steps

Data Loading

Load the dataset into Python using Pandas.

Review data structure and basic statistics.

Exploratory Data Analysis (EDA)

Analyze loan approval vs. rejection trends.

Explore relationships between income, credit history, and approval status.

Create visualizations to identify patterns and anomalies.

Data Cleaning

Handle missing values and duplicates.

Correct data types and standardize categorical variables.

Prepare clean data for SQL and dashboarding.

SQL Analysis

Store cleaned data in a relational database.

Write SQL queries to calculate:

Loan approval rates

Customer segmentation

Key performance metrics

Dashboard Development

Build an interactive Power BI dashboard.

Display KPIs, trends, and filters for deeper analysis.

Reporting & Presentation

Create a summary report with key findings.

Develop a Gamma-based PPT to present insights to stakeholders.

Dashboard

The Power BI dashboard includes:

Overall loan approval rate

Approval trends by income, credit history, and loan type

Customer segmentation insights

Interactive filters for analysis

Results & Insights

Identified key factors influencing loan approval decisions.

Highlighted trends related to credit history and income levels.

Provided data-driven insights to support banking and risk assessment decisions.
Conclusion

This project showcases a complete banking data analytics pipeline, from raw loan application data to executive-ready dashboards and presentations. It reflects practical skills expected from data analysts working in finance and banking domains
